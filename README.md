회고

이번 프로젝트를 진행하면서 고려해야할 부분이 많다는 걸 느꼈다.
처음 설계 부분부터 ui/ux를 고려를 해서 필요한 기능이 들어갈 공간을 고려해야 한다고 생각했다.
회원을 등록하고 그 회원이 로그인해서 책을 빌리면 대여 현황에 넣으려고 했지만 코드를 짜면서 오류발생으로 포기...
로그인하면 개인의 정보만 나오길 원하는데 어떤식으로 진행해야 할지 몰라서 포기...
이후 원하는 기능을 넣으려면 많은 부분을 수정해야할거 같은 느낌이 들어서... (아니면 처음부터 다 고려해서 새로짜야함...)
이런 과정을 통해 처음 시작할때 부터 계획단계가 가장 중요하다고 생각했다. 

아직 1달밖에 되지않아 프로젝트를 짜는 부분이 많이 부족하다고 느끼지만 코딩의 세계의 흥미로움을 느끼고 있어서 
다음에는 좀더 완성도있는 프로젝트를 만들도록 노력해봐야겠다.
